Introduction

What is the essential story being told by your site and what type of structure did you choose to implement.

My website is a travel magazine that provides information about traveling around Cape Town. I chose to go with 5 pages that allowed the users to not only get information about Cape Town but also interact with the site in various ways.

Inspiration

State 3 things that have inspired you when creating your website. These could include guest speakers, other websites, artists, developers etc.

  1. Growing up in Cape Town, I have been fortunate enough to experience all its glory and I wanted to share that in my website.

  2. I was inspired by a website called the insideguide.co.za, a South African online news magazine that I use to work for as an SEO Specialist

  3. When creating my website I looked at unsplashed for inspiration, I also researched international award winning travel website and wanted to incoporate various elements of HTML, CSS and JS I have learnt from the course

Accessibility

State 3 ways in which your site is accessible.

  1. My website has a simple HTML structure, all alt tags have releveant information for example if an image was to not load the alt tag would be able to be read by the screen reader.

  2. Another accessibility feature when using testing the pages I recieve a high score on my light house accesibility report.

  3. I have included a sticky accesibility button on the bottom right of all pages, that when clicked adds a gray scale to the page and all links are underlined.

Usability

State 3 ways in which you have considered usability in your site.

  1. I made sure that my website was responsive

  2. I made sure that my website followed normal structural sites so that it is easy for my users to navigate. As well as using generic icons.

  3. I made ssure that my styling, images and font i.e. the UI and UX of the site was for easy reading and interactive so that the user did not get bored.

Learning

State 3 things you had to learn/find out by yourself when creating your site, and describe how you did that .

  1. I learnt how to embed media files such as video and audio files into my website.

  2. I learned how to structure different files and leave relevant comments so that its easy for me and other developers to work on the site.

  3. I learned how to use multiple js libraries, for example for my gallery and FAQ's. JS is amazinggly versatile.

What aspect(s) of your work do you think worked well and why?

I beleive I got gall the key aspects of my website up and running. Usefull information, HTML, CSS and JS. I also built it so that it is easy for me to add more features and articles. I.e. Modularity.

What aspect(s) of your work could be improved, and how might you do things differently another time?

I would have liked to have added more features to my website such as a booking system, a newsletter and more forms as well as ensure all my links took you places. I would also have liked to have added more content, images and videos to my website.
